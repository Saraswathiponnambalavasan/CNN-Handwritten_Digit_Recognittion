# CNN-Handwritten_Digit_Recognittion
Handwritten digit recognition using a Convolutional Neural Network (CNN) with TensorFlow and Keras.
## Project Overview
This project implements a Convolutional Neural Network (CNN) using TensorFlow and Keras to recognize handwritten digits from the MNIST dataset.

## Features
- Loads the MNIST dataset
- Preprocesses image data
- Builds a CNN model
- Trains the model
- Evaluates test accuracy

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset
MNIST handwritten digit dataset containing 60,000 training images and 10,000 testing images.

## How to Run

Install the required packages:

```bash
pip install tensorflow numpy matplotlib
```

Run the project:

```bash
python training.py
```
